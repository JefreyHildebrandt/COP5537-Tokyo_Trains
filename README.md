**Running the Code:**
* locally dev
```$xslt
node and npm must be installed on the system
cd into the directory with package.json
npm install
npm run dev 
go to localhost:8080
```
  
* production
```$xslt
node and npm must be installed on the system
cd into the directory with package.json
npm run build
node server.js
go to localhost:5000
```
